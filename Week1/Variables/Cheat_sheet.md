# Variables (The Boxes)
A variable is just a box with a name attached to it. You put data inside the box.

- Rule 1: Names must use `snake_case` (e.g., `player_score`, not `Player Score` or `playerScore`).
- Rule 2: Names cannot start with a number.

## Data Types (The Shapes of Data)
- String (`str`): Text. Always wrapped in quotes. `"Hello"`, `"12345"`
- Integer (`int`): Whole numbers. No quotes. `42`, `-5`
- Float (`float`): Decimals. `3.14`, `0.99`
- Boolean (`bool`): `True` or `False`. Must be capitalized. `True`, `False`

## Essential Commands
- `print(item)`: Prints the item to the terminal so you can see it. Tip: You can print multiple things by separating them with commas: `print("Score:", 100)`
- `type(item)`: Tells you what data type is inside the variable.

### Note
In python, putting a '`#`' in front of a text turns it into a comment. A comment is not part of the code. We use comments to describe things in the code without affecting it.
```py
# This is a comment
This is not a comment
```
