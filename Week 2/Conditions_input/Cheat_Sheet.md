
# Python Logic: Week 2 Cheat Sheet

**Comparisons (Asking True/False Questions)**
* `==` : Is equal to (Notice there are **TWO** equals signs. One `=` is for putting things in boxes, two `==` is for asking a question).
* `!=` : Is NOT equal to
* `>` / `<` : Greater than / Less than
* `>=` / `<=` : Greater than or equal to / Less than or equal to

**Compound Logic**
* `and` : BOTH sides must be True
* `or` : ONLY ONE side needs to be True

**The `input()` Function**
* `answer = input("Ask a question: ")` pauses the script and waits for the user to type.
* **CRITICAL:** `input()` ALWAYS gives you a String (`str`). If you want to do math with their answer, you must turn it into an Integer first: `age = int(input("How old are you? "))`

**If / Elif / Else Statements**
* Python uses spaces (indentation) to know what code belongs inside the `if` block.
* Don't forget the colon (`:`) at the end of the statement!
